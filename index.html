Stage en recherche d'ingénierie biomédicale (mai 2025 - janvier 2026)

  • Développement d’un logiciel MATLAB pour analyser automatiquement des courbes biomécaniques : détection de pics, calculs, export Excel, interface utilisateur.
  • Création d’un VI LabVIEW pour visualiser et enregistrer en temps réel les données de 3 plateformes de force synchronisées.
  • Conception 3D sous SolidWorks de boîtiers pour Arduino et potentiomètre : optimisation des dimensions et assemblages.
    • L'objectif était faire un code Matlab capable d'analyser les courbes de manipulations données par une professeure du département de chiropratique. La courbe de manipulation est une manière de décrire l'action précise et contrôlée d'une manipulation chiropratique, qui vise à rétablir le mouvement articulaire et la fonction du corps. Le chiropraticien utilise ses mains pour appliquer une force rapide et dirigée sur une articulation qui a une mobilité limitée ou est bloquée. Le code devait détecter automatiquement 3 points sur la courbe : le maximum le peak force, le peak preload et le début du Thrust initiation.
